# 🦖 Dino Runner — Бесконечный бег динозаврика

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue)](https://ваш-логин.github.io/dino-runner/)

Классическая игра «Динозаврик» из браузера Chrome, но с улучшенной графикой, адаптивным дизайном и плавным управлением. Проект полностью на HTML/CSS/JS, без внешних зависимостей.

![Превью игры](https://via.placeholder.com/800x250?text=Dino+Runner+Preview)  
*(Здесь можно добавить скриншот или гифку игры)*

## 🎮 Игровой процесс

Вы управляете бегущим динозавриком, который должен перепрыгивать кактусы. С каждым прыжком счёт увеличивается, а игра постепенно ускоряется. Цель — продержаться как можно дольше и побить свой рекорд.

## 🕹️ Управление

| Действие | Клавиши (ПК) | Касание (телефон) |
|----------|--------------|-------------------|
| Прыжок   | `Пробел` / `↑` (стрелка вверх) | Тап по экрану / canvas |

## ✨ Особенности

- **Адаптивный интерфейс** — игра отлично выглядит на любых устройствах (ПК, планшет, телефон).
- **Плавная анимация** прыжков и бега.
- **Сохранение рекорда** в локальном хранилище браузера.
- **Увеличение сложности** — скорость игры растёт с каждыми 250 очками.
- **Динамический спавн препятствий** — кактусы появляются с разными интервалами.
- **Визуальные улучшения**:
  - Градиентное небо.
  - Детализированный динозаврик с анимацией ног.
  - Кактусы с колючками.
  - Облака, плывущие по фону.
  - Эффект земли с камешками.
